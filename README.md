## About

Happy Cake is a landing page for a custom cake bakery based in Vinnytsia. Features a product catalog with order buttons, a testimonials slider, photo gallery and a modal consultation form. Built with pure HTML, SCSS and vanilla JavaScript.

🔗 **Live:** [happy-cake](https://vergos1.github.io/happy-cake/)

## Sections

| Section | Description |
|---|---|
| **Hero** | Full-screen background with CTA button |
| **About Us** | Company info and values |
| **Features** | Quality, exclusivity and speed highlights |
| **Catalog** | Wedding, children, holiday and anniversary cakes |
| **Reviews** | Testimonials slider with dot navigation |
| **Gallery** | 8-image photo grid |
| **Modal** | Consultation request form |

## Features

- **Reviews slider** — fade animation with dot and arrow navigation
- **Order modal** — consultation form with open/close animation
- **Responsive** — adaptive layout for mobile and desktop
- **Smooth scroll** — anchor navigation with CSS scroll-behavior

## Getting Started

No build step required — open directly in browser.

```bash

git clone https://github.com/Vergos1/happy-cake.git

open index.html
```
